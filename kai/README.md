# Exercícios de Java
 <p style = "font-size: 16px;">Conjunto de exercícios propostos para reforçar conceitos base da programação com a linguagem Java.</p>

| Conceitos Utilizados |
| :--- |
| POO (Abstração, encapsulamento, polimorfismo e herança). |
| Classes abstratas, criação e implementação de métodos, reutilização de código e treino de lógica de programação. |

---

|  Atividades Propostas |
| :--- |
|1. criar exercício para conversão de celsius para fahrenheit, “com métodos separados” |
|2. criar calculadora de IMC métodos para calcular |
| 3. criar um calculadora de descontos, com método separa imprimindo com classe scanner para os preços e descontos |
| 4. criar um conversor de Unidades metros para centímetros, centímetros para metros, quilômetros para metros, metros para quilômetros e metros para milhas. |
| 5. Criar um Validador de Dados - Criar método que verifica se número é par ou ímpar, Criar método que verifica se número é positivo, negativo ou zero, Criar método que verifica se ano é bissexto |
